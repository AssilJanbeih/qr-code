# qr-code
This project is a mobile app build using flutter and dart language. <br><br>
Features: <br>
-Scan QR code <br>
-View QR code link<br> 
-Save QR link to a local database on mobile using MySQL database.
<br><br>
Packages used: <br>
-qr_code_scanner <br>
-url_launcher <br>
-sqflite <br>
-path_provider
